Class: Encounter

Encounter(new_encounter)

Primitive class that represents a patient encounter, which is defined as a contact with the patient. If it is a hospital admission, admit date and discharge date are self explanitory. If it is a non-admit encounter, the admit date/time will be used as the encounter time Designed to be extended by browser (frontend) and node.js (backend) specific models
